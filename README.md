# Customer Support Performance Dashboard

This Power BI project analyzes customer support data to track KPIs such as:
- Total Tickets
- Resolved Tickets
- Escalations
- SLA Breach%
- CSAT Score
- Average Resolution Time

Insights:
• EU region has highest SLA breach
• Call channel receives most tickets
• Some agents need resolution speed improvement

Tools Used:
• Power BI
• DAX for Calculations
