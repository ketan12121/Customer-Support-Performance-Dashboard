# Customer Support Performance Dashboard

This Power BI project analyzes customer support data to track KPIs such as:

- Total Tickets  
- Resolved Tickets  
- Escalations  
- SLA Breach %  
- CSAT Score  
- Average Resolution Time  

## Insights

From the dashboard, we can see:

- The **EU region** has the highest SLA breach %.  
- The **Call channel** receives the most tickets.  
- Some agents have **higher average resolution time**, indicating a need for performance improvement or training.

## Tools Used

- Power BI  
- DAX (for measures and KPIs)

## How to View the Dashboard

1. Download `Customer Support Dashboard.pbix` from this repository.  
2. Open it using **Power BI Desktop**.  
3. Use the slicers (Date, Region, Channel) to explore the data interactively.
